## Current Implementation

### String Permutation using Backtracking

This project generates all permutations of a given string using
a recursive backtracking approach.

The implementation focuses on understanding how recursive search
explores the solution space step by step.

---

## Algorithm Overview

Backtracking works by:

1. Selecting a character that has not been used
2. Placing it into the current position
3. Recursively continuing the construction
4. Undoing the choice to explore other possibilities

A boolean array is used to track visited characters during recursion.
